BERT原理

BERT是基于Transformer的双向编码模型，它通过Masked Language Model和Next Sentence Prediction进行预训练。

Masked Language Model是通过将一些输入的单词随机遮盖掉，然后让模型来预测这些被遮盖的单词，从而使得模型能够学习到上下文相关的单词表示。

而Next Sentence Prediction则是通过输入两个句子，并让模型来判断这两个句子是否是连续的，从而使得模型能够学习到句子间的关系。
虽然BERT是基于transformer的，但是它只使用了transformer的encoder部分，它的整体框架是由多层transformer的encoder堆叠而成的。

模型输入

BERT模型的输入就是wordpiece token embedding + segment embedding + position embedding.如下图：

  1. wordpiece embedding：单词本身的向量表示。WordPiece是指将单词划分成一组有限的公共子词单元，能在单词的有效性和字符的灵活性之间取得一个折中的平衡。
  2. position embedding：将单词的位置信息编码成特征向量。因为我们的网络结构没有RNN 或者LSTM，因此我们无法得到序列的位置信息，所以需要构建一个position embedding。构建position embedding有两种方法：BERT是初始化一个position embedding，然后通过训练将其学出来；而Transformer是通过制定规则来构建一个position embedding
  3. segment embedding：用于区分两个句子的向量表示。这个在问答等非对称句子中是用区别的。

BiLSTM（双向长短期记忆网络），是为了增强文本序列中“顺序结构”的建模。补充时间依赖。
