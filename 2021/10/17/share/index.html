垂直居中的多种方法

2021-10-17

垂直居中水平居中的几种方法

方法一：flex布局法

1.在父级盒子中添加flex css样式
display:flex
justify-content:center;
flex-direction:column;/row;/*垂直或水平*/
2.垂直居中
display: flex;
align-items: center;

方法二：绝对定位法

1.绝对定位下top left right bottom 都设置0（水平垂直居中）
position:absolute;
left:0;
top: 0;
bottom: 0;
right: 0;
margin: auto;
2.
Transform:包括旋转rotate、扭曲skew、缩放scale和移动translate以及矩阵变形matrix。translate(x,y),它表示对象进行平移，当值为负数时，反方向移动物体.translate可以在不知道宽高的情况下进行居中，tranlate（）函数中的百分比是相对于自身宽高的百分比，所以能进行居中。
div{
    background:red;
    position: absolute;
    left:50%;
    top:50%;
    transform: translate(-50%, -50%);
}

Left:50%;top:50%相当于把盒子的左上角 定位在body的中心。

设置transform:translate(-50%,-50%),即延X轴和延Y轴均移动50%，对比原图位置，移动后的图如下所示，

3.确定了当前div的宽度，margin值为当前div宽度一半的负值
div{
  width:600px;
  height: 600px;
  background:red;
  position: absolute;
  left:50%;
  top:50%;
  margin-left:-300px;
  margin-top:-300px;
}
4.　table-cell实现水平垂直居中: table-cell middle center组合使用
<div class="table-cell">
    <p>我爱你</p>
</div>
.table-cell {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
    width: 240px;
    height: 180px;
    border:1px solid #666;
}
